2. & 3. prednáška

Do záhlavia písomky napíšte dnešný dátum rímskymi číslicami.

  • dni a mesiace prepisujeme podľa rímskej sústavy:
    • I = 1,
    • V = 5,
    • X = 10,
    • L = 50,
    • C = 100,
    • D = 500,
    • M = 1000.

Postup:

  1. Prevod HEX → BIN (každá hex cifra = 4 bity).
  2. Binárny súčet s prenosmi (ako na snímkach 13–14).
  3. Prevod BIN → OCT (3 bity = 1 číslica v osmičkovej sústave).

(Uvedené konkrétne čísla doplníš podľa zadania — postup je vždy rovnaký.)

Príklad:

Spočítaj: A3₁₆ + 2F₁₆

  1. HEX → BIN
    • A = 1010
    • 3 = 0011
    • 2 = 0010
    • F = 1111

    A3₁₆ = 1010 0011₂
    2F₁₆ = 0010 1111₂

  2. Sčítanie

      1010 0011 
    + 0010 1111 
    ------------
      1101 0010₂
  3. Prevod výsledku do OCT

    110 100 010 = 6 4 2 → 642₈

Výsledok: 642₈

Postup:

  1. Prevod OCT → BIN (3 bity = 1 osmičková cifra).
  2. Násobenie 4 = aritmetický posun doľava o 2 bity (<<2)
    • Zodpovedá snímke 9–10 o posunoch.
  3. Prevod BIN → HEX (skupiny po 4 bitoch).

Príklad:

Číslo: 157₈

  1. OCT → BIN
    • 1 = 001
    • 5 = 101
    • 7 = 111
    157₈ = 001 101 111₂ = 001101111₂
  2. Násobenie 4 = posun o 2 bity doľava

    001101111 << 2 = 110111100₂
  3. BIN → HEX

    0110 1111 00
    = 0011 0111 1100
    = 3 7 C → 37C₁₆

Výsledok: 37C₁₆

Pravidlá sčítania BCD:

  • Sčítame binárne ako normálne (bitovo).
  • Ak výsledok cifry > 9 (1001₂), alebo nastal prenos → pripočítame 6 (0110₂).
  • Prenos pokračuje do ďalšej desiatkovej cifry.

(Doplníš čísla podľa zadania.)

Príklad:

Sčítaj: 47 + 28 v BCD.

BCD:

  • 4 = 0100
  • 7 = 0111
  • 2 = 0010
  • 8 = 1000
  1. Sčítanie jednotiek:

      0111 
    + 1000 
    -------
      1111₂ = 15₁₀ → neplatná BCD cifra (>9)
  2. Pripočítať 6 (0110₂):

      1111 
    + 0110 
    -------
    1 0101 (5 + carry 1)
  3. Sčítanie desiatok:

      0100 
    + 0010 
    + 0001 (carry)
    ----------
      0111 = 7

Výsledok v BCD: 0111 0101 (75)

Postup:

  • Deliť číselnú hodnotu postupne dvomi a zapisovať zvyšky odzadu.
  • Toto je presný postup pre binárnu reprezentáciu v prednáške (snímka 13).

Príklad:

Preveď: 45₁₀

  1. Delenie dvomi:
    2. Delenie Zvyšok
    45 / 2 = 22 1
    22 / 2 = 11 0
    11 / 2 = 5 1
    5 / 2 = 2 1
    2 / 2 = 1 0
    1 / 2 = 0 1
  2. Zvyšky odzadu:

45₁₀ = 101101₂

Postup:

  • Pre druhé číslo (odčítanec) urobíme doplnkový kód:
  1. Inverzia bitov (NOT)
  2. Pripočítanie 1
  3. Pripočítame ku prvému číslu.
  4. Ignorujeme prenos za najvyšší bit.

Tým získame výsledok odčítania.

Príklad:

Vypočítaj: 10110₂ – 01101₂

  1. Doplnkový kód odčítanca (01101):
    • Invertovanie:
      • 01101 → 10010
    • Pripočítať 1:
      • 10010 + 1 = 10011
  2. Sčítanie:

      10110  
    + 10011  
    ---------
    1 01001  

Odstránime pretečený bit:

Výsledok: 01001₂ = 9

Postup:

  • Pre každý bit násobiteľa:
    • ak je bit 1 → pripočíta sa posunutý súčin,
    • ak je bit 0 → ignoruje sa.
  • Ide o „shift-and-add“ algoritmus.

Príklad:

Vypočítaj: 1011₂ × 110₂

      1011 
×     110  
-----------
      0000   (posun ×0)
     1011    (posun ×1)
+   1011     (posun ×2)
------------
   1000010₂ 


Výsledok: 1000010₂

  • 1 kilobajt (kB) = 1000 bajtov (dekadická sústava)
  • 1 kibibajt (KiB) = 1024 bajtov (binárna sústava)

Rozdiel:

  • 1024 − 1000 = 24 bajtov → Kibibajt je väčší o 24 bajtov.

Status register obsahuje napríklad:

  • Carry bit (C) - pretečenie do vyššieho bitu pri sčítaní.
  • Overflow bit (V / O) - znamená prekročenie rozsahu znamienkového čísla.
  • Zero flag (Z) - výsledok operácie je nula.
  • Negative flag (N) - výsledok je záporný
  • Semiotika - veda o znakoch ako takých, ich význame, interpretácii, symboloch.
  • Lingvistika - veda o ľudskom jazyku, jeho gramatike, štruktúre, fonetike.

Semiotika je širšia - lingvistika je jej podmnožina.

Postup:

  • Prefix: operátor ide pred argumenty
  • Napr. infix: (5 – 6) / 7
    • → prefix: / - 5 6 7

Paritný bit je nastavený takto:

  • Ak je počet jednotiek v dátovom slove párny, paritný bit = 1
  • Ak je počet jednotiek nepárny, paritný bit = 0

Chyba nastane, ak paritný bit nezodpovedá počtu jednotiek v riadku. → Vyberieš riadok, kde sa parita nezhoduje.

Príklad:

  • Parita (párna):
    • ak je počet jednotiek v dátach párny → paritný bit = 1
    • ak nepárny → paritný bit = 0
    • Dáta Počet 1 Paritný bit Správne?
    101011 4 1 ✓
    111000 3 1 ✗ CHYBA
    010111 4 1 ✓

Chybný riadok: 2.
